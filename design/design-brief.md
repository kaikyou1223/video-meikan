# video-meikan デザイン指示書

## 1. プロジェクト概要

### サービス名
video-meikan（動画名鑑）

### コンセプト
AV女優×ジャンルの組み合わせに特化したデータベースサイト。
検索流入がメインのため、**情報の見つけやすさ・回遊性・表示速度**を最優先とする。

### ターゲットユーザー
- 30〜40代の男性
- 「女優名 ジャンル」で検索してたどり着くユーザー
- スマートフォンからのアクセスが中心

### 参考サイト
- https://av-wiki.net/av-actress/yukina-mafuyu/
  - 落ち着いたトーン、カード型の作品表示、パンくず＋ページネーション

---

## 2. デザインの方向性

### 2.1 トーン & ムード

| 項目 | 方向性 |
|------|--------|
| 全体の印象 | 清潔感のあるダークトーン。アダルト感を出しすぎず、「データベース / カタログ」としての信頼感 |
| 配色 | ダークグレー背景（`#1a1a2e` 系）＋ アクセントカラー（ゴールドまたはブルー系） |
| フォント | ゴシック体ベース。見出しは太め、本文は読みやすさ重視 |
| 余白 | ゆとりをもたせ、情報が詰まりすぎないようにする |
| 角丸 | カード・ボタンは `4〜8px` 程度の角丸 |

### 2.2 避けたいデザイン
- ギラギラしたアダルトサイト感（派手なピンク・赤の多用）
- 文字が小さすぎて読みにくいレイアウト
- 広告が多すぎてコンテンツが埋もれる印象

## 3. ページ別デザイン仕様

### 3.1 共通レイアウト

```
┌──────────────────────────────────────┐
│ ヘッダー                              │
│  ロゴ（左）    ナビ（右）              │
├──────────────────────────────────────┤
│ パンくずリスト                         │
├──────────────────────────────────────┤
│                                      │
│ メインコンテンツ                       │
│                                      │
├──────────────────────────────────────┤
│ フッター                              │
│  サイト情報 / リンク                    │
└──────────────────────────────────────┘
```

#### ヘッダー
- 左：サイトロゴ（テキストロゴでOK「video-meikan」）
- 右：ナビゲーション（名鑑TOP / ジャンル一覧）
- モバイル時はハンバーガーメニュー
- 背景色：メインカラーより濃いダーク色
- 高さ：PC 60px / SP 50px

#### パンくずリスト
- 表示例：`TOP > 瀬戸環奈 > NTR`
- ヘッダー直下に1行で表示
- フォントサイズ小さめ（12〜13px）

#### フッター
- サイト名、コピーライト
- 利用規約・プライバシーポリシーへのリンク
- シンプルに1〜2行程度

#### 仕様

  - サムネイル画像（アスペクト比 3:4、上部）

#### 仕様

##### プロフィールセクション
- 左：女優サムネイル画像（PC: 150×200px / SP: 100×133px）
- 右：女優名（h1、20〜24px、太字）＋ 作品数
- 背景をカードっぽく少し浮かせる（シャドウ or 背景色差）

##### ジャンルタグ一覧
- 横並びのタグUI（`display: flex; flex-wrap: wrap;`）
- 各タグはアンカーリンク（ページ内の該当ジャンルセクションへスクロール）
- タグのスタイル：ピル型（角丸16px）、アクセントカラーのボーダー
- ホバー時：背景色がアクセントカラーに変化

##### ジャンル別カルーセルセクション
- **セクション見出し**：ジャンル名（h2、18px）＋ 作品数 ＋ 「もっと見る →」リンク（右寄せ）
- **カルーセル**：横スクロール（`overflow-x: auto`、CSS scroll-snap推奨）
  - PC：4〜5枚表示 ＋ 左右スクロール矢印ボタン
  - SP：2枚表示 ＋ スワイプ操作
- **作品カード（カルーセル内）**：
  - サムネイル画像（アスペクト比 16:9、DVD/動画ジャケット比率）
  - タイトル（1〜2行、14px、`text-overflow: ellipsis` で省略）
  - 幅：PC 200px / SP 150px
- **「もっと見る」リンク**：ジャンル別作品ページへ遷移
- **セクション間の広告枠**：3〜4ジャンルごとに1箇所、広告バナー枠（横幅100%、高さ90〜250px）

**目的**：特定の女優×ジャンルの全作品を一覧で見る。アフィリエイト遷移を促す。

#### 仕様

##### ページタイトル
- h1：「{女優名}の{ジャンル名}作品一覧」（20〜24px、太字）
- 作品件数をサブテキストまたはバッジで表示

##### 作品カード（横型・リスト表示）
- **レイアウト**：横型カード（画像左 + 情報右）
  - PC：画像 240×135px（16:9）＋ 右に情報
  - SP：画像 120×68px（16:9）＋ 右に情報（または縦積みに切替）
- **表示項目**：
  - タイトル（太字、14〜16px、最大2行で省略）
  - 発売日（12px、グレー）
  - レーベル名（12px、グレー）
  - 「作品を見る →」ボタン（アフィリエイトリンク）
- **カードのインタラクション**：
  - ホバー時：背景色がわずかに明るくなる
  - CTAボタン：アクセントカラー、ホバーで色変化
- **広告枠**：3〜5作品ごとに1箇所挿入

##### ページネーション
- ページ下部に配置
- 現在ページをアクセントカラーでハイライト
- 前へ / 次へ ＋ ページ番号（最大5〜7個表示）
- 1ページあたり20作品

- ファーストビューに広告を詰め込みすぎない（コンテンツが見える状態を維持）
- 広告枠はコンポーネント化し、後から位置調整できるようにする

## 5. レスポンシブ対応

### ブレークポイント

| 名称 | 幅 | 備考 |
|------|-----|------|
| SP | 〜767px | スマートフォン（メイン） |
| タブレット | 768〜1023px | - |
| PC | 1024px〜 | 最大幅 1200px で中央寄せ |

### SP対応の重要ポイント
- **カルーセル**：スワイプ操作前提、矢印ボタンは非表示
- **女優カード**：2カラムグリッド
- **作品カード（ジャンル別ページ）**：画像上＋情報下の縦積みも検討
- **タッチターゲット**：ボタン・リンクは最低44×44px確保
- **フォントサイズ**：本文14px以上を確保

## 8. 補足・注意事項

- 画像はすべて外部URL（FANZA/MGS）から取得するため、画像サイズは制御不可 → **アスペクト比固定＋`object-fit: cover`** で統一する前提でデザインしてください
- サイトのジャンル特性上、作品サムネイルはアダルト画像になります。デザインカンプではダミー画像（グレーボックス等）で問題ありません
- 実装はPHP（素のHTML出力）で行うため、特殊なフレームワーク依存のコンポーネント設計は不要です
- カルーセルはCSS scroll-snap ＋ 軽量JSで実装予定のため、複雑なアニメーションは不要です
